Open source projects

We maintain a number of open-source projects on GitHub. Feel free to fork these repositories, and if you wish to submit an enhancement to a project, submit a pull request. You may use GitHub Desktop if you are unfamiliar with git.

Binary.com is actively developing these open-source applications:

  • binary-static – the Binary.com website itself.
  • mobile – a mobile app for Android and iOS.
  • binary-next-gen – a next-generation UI; both a mobile application and a desktop application. See app.binary.com.
  • webtrader – a desktop application. See Binary Webtrader.
  • binary-bot – an automated strategy builder. See Binary Bot.
  • developers.binary.com – this website itself.

and the following open-source libraries:

  • binary-utils – common utility functions for use in all Binary.com apps.
  • binary-live-api – a high-level API library, to make using the Binary.com API easier.
  • binary-charts – reusable binary options charting component. See showcase.
  • binary-style – Style guide for Binary.com projects. See guide.
  • binary-indicators – technical analysis indicators library
  • imacros – compatibility layer for imacros scripts

Binary.com's open-source projects are typically published via the Github Pages service.
